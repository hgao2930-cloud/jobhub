## 二、项目经验

### 项目 1：JobHub 求职管理系统（传统前端方向）

```
项目描述：
基于 Vue 3 + Vite 开发的求职管理 SPA，支持岗位投递记录跟踪与数据统计。

技术栈：Vue 3 + Vue Router + Axios + json-server

主要工作：
- 使用 Vue 3 Composition API（ref、computed、onMounted）构建页面逻辑
- 通过 Vue Router 实现嵌套路由（布局壳 → 子页面）和路由守卫（未登录拦截）
- 使用 Axios 封装 HTTP 请求，配合 json-server 实现前后端分离开发
- 实现用户登录/注册认证、岗位 CRUD、搜索筛选、分页等功能
- 使用 Vite 作为构建工具，配置 @ 路径别名和 API 代理
```

### 项目 2：杭州智慧旅游 WebGIS 交互地图（WebGIS 方向）

```
项目描述：
基于 OpenLayers 开发的杭州景点交互地图，实现地图可视化、绘图交互与空间数据管理。

技术栈：OpenLayers 10 + 原生 JavaScript（ES6+ 模块化）+ Vite

主要功能：
- 多底图切换（高德 / OSM），按景区等级彩色图钉标注景点
- 景点筛选（按类别/名称）与地图定位
- 绘图工具：绘制点/线/面/圆，支持编辑、选中、删除
- 数据持久化：绘图数据自动保存至 localStorage
- GeoJSON 格式导入/导出、测距/测面积
- 空间查询：绘制范围查询区域内景点

代码模块按功能拆分：map.js / draw.js / spots.js / storage.js / measure.js / query.js
```
